To enable wayland:
  - sudo nvim /etc/gdm3/custom.conf
  - Set WaylandEnable=true
  - reboot and select wayland and login :)
  - https://www.reddit.com/r/pop_os/comments/1d4l49t/how_do_you_enable_wayland_in_pop_os_2024/

To trim a video, use ffmpeg:
  # grabs the first 10 minutes of your video
  ffmpeg -i ./your_video.mp4 -ss 00:00:00 -to 00:10:00 -c copy output.mp4

Map a folder from linux PC to Windows PC:
  - Open file explorer on windows and right click on 'This PC' and click on 'Map network drive'
  - type 'ip a' in linux terminal to get ip address of linux pc
  - type in \\[ip address of linux pc]\Shared Folder Name
  - type 'sudo nvim /etc/samba/smb.conf' in the terminal and go the bottom and type this:
      [LinuxShare]
        path = /home/your_username/Shared
        browseable = yes
        writable = yes
        read only = no
        guest ok = yes
        valid users = leo_zhang

replace the values with your desired ones
also the name can be anything and not strictly "LinuxShare"
whatever name you give it will be what you map to in windows
In this example what you would map to in windows would be: \\[ip address of linux pc]\LinuxShare
